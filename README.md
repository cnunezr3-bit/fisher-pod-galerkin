# POD–Galerkin Reduced-Order Models for the Optimal Control of the Fisher Equation

This repository contains the Python implementation accompanying the paper

> POD–Galerkin reduced-order models for the optimal control of the Fisher equation.

## Requirements

- Python 3.10+
- NumPy
- SciPy
- Matplotlib

Install the required packages with

```bash
pip install -r requirements.txt
```

## Running the code

Run the main numerical example

```bash
python main.py
```

Additional numerical experiments are available through

- sensitivity_diffusion.py
- sensitivity_snapshots.py
- sensitivity_coupled_vs_separate.py
- sensitivity_coupled_vs_separate_lowD.py
- sensitivity_out_of_training.py
